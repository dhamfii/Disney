# Disney
1. Project Title / Heading

Disney Parks Visitor Insights Dashboard

2. Short Description / Purpose

This project is a simple and interactive Power BI dashboard built to explore different insights related to Disney parks.
The main goal was to understand trends like visitor flow, ticket pricing, ratings, and overall patterns using real-world data.
It is designed as a learning project to practice data cleaning, visualization, and dashboard building.

3. Tech Stack

⦁	Power BI (PBIT Template)
⦁	Excel / CSV for data preparation
⦁	Power Query (M language)
⦁	DAX for calculated measures

4. Data Source

The dataset used in this project was downloaded from Kaggle, publicly available for learning and analysis purposes.

5. Features / Highlights

⦁	Clean and simple dashboard layout
⦁	Visitor trends across different Disney parks
⦁	Ticket price variations and comparisons
⦁	Park ratings and popularity insights
⦁	Interactive slicers and filters for easy exploration
⦁	Designed with a theme inspired by Disney visuals
⦁	Beginner-friendly project showcasing Power BI skills
